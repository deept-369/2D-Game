# 🇮🇳 India Cultural Adventure - 2D Game

A 2D adventure game inspired by the rich culture, traditions, and folklore of India. Players travel across **9 different Indian states**, experiencing unique stories, landmarks, festivals, and challenges based on each state's heritage.

---

# 🎮 About the Game

This game takes players on a journey through India, where they explore different states and uncover cultural stories inspired by local traditions and legends. Each level is designed to reflect the identity, architecture, art, and lifestyle of a specific state.

---

# 🌟 Features

- Explore **9 Unique Indian States**
- Story-driven gameplay based on regional culture
- Beautiful 2D pixel-art environments
- Interactive NPCs inspired by local communities
- Quests, puzzles, and collectibles
- Educational and entertaining experience
- State-specific landmarks and folklore

---

# 🗺️ States Included

- Odisha
- Gujarat
- Rajasthan
- Maharashtra
- Punjab
- West Bengal
- Tamil Nadu
- Kerala
- Assam

---

# 🎯 Gameplay

Players travel through different states, complete cultural quests, solve puzzles, and learn about India's diverse heritage. Successfully completing a state's challenges unlocks the next region and story.

---

# 🛠️ Built With

- Godot Engine
- GDScript
- TileMap System
- Pixel Art Assets
- Custom Animations
